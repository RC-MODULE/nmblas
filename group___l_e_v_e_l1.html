выполняет поворот массива точек в двумерном пространстве (поворот Гивена)

[ x_i ] = [ c s ] [ x_i ]
[ y_i ] = [ -s c ] [ y_i ]
Если n <= 0 или если c = 1 и s = 0, функция выходит сразу.

[in,out]*Yуказатель на массив, который содержит y координату точки
[in]INCYсмещение между элементами в массиве
[in]Сзначение косинуса угла поворота
[in]Sзначение синуса угла поворота
void nmblas_drotg ( double *  a,
double *  b,
double *  c,
double *  s 
)

вычисяет параметры необходимые для пороворота Гивена

Аргументы
[in,out]*Aуказатель на параметр a, после выполнения a будет переписан параметром r(смотри детальное описание)
[in,out]*Bуказатель на параметр b, после выполнения a будет переписан параметром z(смотри детальное описание)
[out]*Cпосле выполнения будет записан параметр с(смотри делальное описание)
[out]*Sпосле выполнения будет записан параметр s(смотри делальное описание)

drotg вычисляет параметрв для поворота Гивена.
Принимая скаляр а и b,программа вычисляет следущие параметры:

sigma = sgn(a) если |a| > |b|, и sgn(b) в противном случаии;
r = sigma * sqrt( a^2 + b^2 );
c = a / r если r <> 0, иначе 1;
s = b / r если r <> 0, иначе 0;

числа c, s и r будут удовлетворять следующему матричному уровнению:

[ c s ] [ a ] = [ r ]
[ -s c ] [ b ] = [ 0 ]

Sigma не важна для вычислений матрицы поворота Гивена, но важна для
дальнейшего надежного востановления c и s из одного сохраненого числа.
Для этого данная функция также вычисляет z как показано ниже

z = s если |a| > |b|,
z = 1 / c если |b| >= |a| и c <> 0,1
z = 1 если c = 0.

Функция возвращает r, переписывая a и z переписывая b, также возвращая c и s.
Если позже будет необходимость в реконструировании c и s из z это можно будет сделать следующим образом

если z = 1, установить c = 0 и s = 1,
если |z| < 1, установить c = sqrt(1 - z^2) и s = z,
если |z| > 1, установить c = 1 / z и s = sqrt(1 - c^2).

Смотри ``Basic Linear Algebra Subprograms for Fortran Usage'' C. Law-
son, R. Hanson, D. Kincaid and F. Krogh, ACM Transactions on Mathema-
tical Software, 1979, 5(3) pp 308-323, для болеее подробной информации.

функция применяет модифицированный поворот Гивена (modified-Givens rotation), матрица H предается через *PARAM

[ x_i ] [ h_11 h_12 ] [ x_i ]
[ y_i ] = [ h_21 h_22 ] [ y_i ]
Если n <= 0 или если H индентичная матрица, выход из функции происходит сразу

Аргументы
[in]Nпеередает длину векторов.
[in,out]*Xуказатель на вектор X, после выполнения будет переписан результатом
[in]INCXпередает инкремент массива X (шаг между элементами)
[in,out]*Yуказатель на вектор Y, после выполнения будет переписан результатом
[in]INCYпередает инкремент массива Y (шаг между элементами)
[in]*PARAMуказатель на массив размерности не менее 5(double), который содержит матрицу H.
